## Getting Started

First, create a `.env` file and add the following:

```
INSTANT_DB_ADMIN_TOKEN=
VITE_INSTANT_APP_ID=
ENCRYPTION_PASSWORD=
```

### Using Bun

```
bun install
bun run dev
```

### Using NPM

```
npm install
npm run dev
```
